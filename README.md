# Bells

## Purpose

A reference implementation of several libraries integrated with Django.

## Python

- Channels
  - Websocket support
- Daphne
  - Async server
- Django
- Poetry
  - Python dependency management
  - Plus a local package dependency

## JS

- Pixi.js
  - 2D Graphics support
- SolidJS
  - Reactive components
- TypeScript
  - Makes JavaScript much more fun
- Vite
  - Modern bundler
